# 🌍 personal-footprint-map

本项目是一个基于 **ECharts GL**、**Supabase** 和 **Vanilla JS** 构建的地理信息可视化系统，旨在记录并展示本人的足迹。这时本人的第一个项目，也是第一个vibe coding的项目。

---

## 🚀 核心功能清单

- [x] **3D 地球自动旋转与大气层效果**
- [x] **全球国家填色与行政区下钻**
- [x] **中国省市二级填色控制**
- [x] **智能地点搜索与历史定位**
- [x] **年份足迹动态回放 (演示模式)**
- [x] **多维标签筛选与配色定义**
