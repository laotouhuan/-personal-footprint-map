# 足迹地图后续版本待办

本文档用于记录第二版相对 [readme1.md](./readme1.md) 还没有实现的功能，方便下一次对话继续迭代。

## 当前第二版已完成

- 中国地图 GeoJSON 改为优先读取项目内静态文件（`./maps/china/`），外部接口作为备用
- 增加"足迹高亮 / 普通地图"切换按钮
- 增加"省级填色 / 市级填色"切换按钮（市级填色会合并全国所有省份的市级 GeoJSON）
- 增加同行者标签多选筛选，地图只显示选中标签对应的足迹
- 省级填色规则改为按该省"最早足迹"的标签颜色（而非最新）
- 增加省 / 市搜索栏，支持模糊匹配，选中后自动进入对应省份地图
- 增加地点聚合面板，可查看该地点全部历史足迹、删除记录、基于该地点新建足迹
- 本地服务（server.ps1）已支持 `.json` 文件正确返回 `application/json` MIME 类型

## 相比 readme1.md 仍未实现的功能

### 1. 世界地图与多国支持

- 世界地图默认 3D 地球视图（支持鼠标拖拽旋转和缩放）
- 从世界地图点击国家，平滑下钻进入该国 2D 行政图
- 中国以外国家的省级 / 州级地图渲染与足迹填色
- 首屏内置世界地图基础框架数据
- 中国以外国家边界数据的懒加载方案
- 更完整的平滑下钻动画和视角过渡效果

### 2. 权限与安全

- 默认访客严格模式：未登录时仅显示空白基础地图，所有管理控件强制隐藏（目前未登录仍可看到地图结构，但因无数据所以无填色）
- 真正的隐藏式登录入口：目前顶栏"管理员登录"按钮对所有访客公开可见，不符合原始设计中"默认无任何公开登录入口"的要求
- 只读分享链接：通过附带随机分享令牌的 URL（如 `?share=加密字符串`）向朋友展示足迹，且该访客仅有只读权限

### 3. 悬浮提示（Tooltip）行为细节

- 当前 Tooltip 中展示的记录受同行者筛选器（`activeCompanionFilters`）影响，即被过滤掉的标签记录不会出现在悬浮提示中；按 readme1.md 原始设计，Tooltip 应展示**该地区的所有访问记录**，不受当前筛选状态影响

### 4. 部署与工程化

- GitHub Pages 正式部署流程
- 针对 GitHub Pages 的部署说明文档（环境变量、路径、MIME 类型等注意事项）
- 更明确的本地环境初始化说明（依赖、首次运行步骤）

## 需要修改和增加的新功能

- 加上台湾省的市级行政区的内容
- 将标签中的“家人”改为“父母”，添加一个新的标签名为“阿姨”
- 如果某地有多个相同时间、不同标签的"最早足迹"，则该地的填色需要有已被筛选的标签的颜色的体现（渐变色、分成几块均可，使得页面美观即可）。比如说有三个“最早足迹”，筛选了三个标签就要有这三个标签的颜色的体现，筛选了两个个标签就要有这两个标签的颜色的体现。
- 足迹簿请加上时间轴
- 足迹簿和搜索栏中的足迹，除了删除还需要有修改足迹的功能，能够修改足迹的具体内容
- 将筛选的标签隐藏，必须点击一处后才能筛选具体的标签
